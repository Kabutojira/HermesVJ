Generate a self-contained p5.js instance-mode sketch.

Requirements:
- Export `default function sketch(p)`.
- Use `p.createCanvas(p.windowWidth, p.windowHeight)` in `setup`.
- Keep the sketch responsive in `windowResized`.
- No DOM access beyond p5 canvas creation.
- No network calls, no remote assets, no eval, no dynamic imports.
- Prefer procedural visuals.
- Output code only.
